Passos para configurar

  1. Verificar as credenciais do windows
  2. Configurar o nome do email do git
  3. Listar as configuraçoes

Comandos:

verificar:
git config --list

Configuração:

git config --global user.name "seu nome"
git config --global user.email "seu email"
